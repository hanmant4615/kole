## üîß Technical Details

### Translation System Architecture:

```
translations/
‚îú‚îÄ‚îÄ index.ts          # Central merger (exports allTranslations)
‚îú‚îÄ‚îÄ common.ts         # 100+ shared keys
‚îú‚îÄ‚îÄ home.ts           #  80+ home page keys
‚îú‚îÄ‚îÄ schemes.ts        # 200+ schemes keys
‚îú‚îÄ‚îÄ about.ts          # 550+ about section keys
‚îú‚îÄ‚îÄ departments.ts    # 850+ department keys
‚îú‚îÄ‚îÄ services.ts       # 120+ services keys
‚îú‚îÄ‚îÄ news.ts           #  80+ news keys
‚îú‚îÄ‚îÄ gallery.ts        #  35+ gallery keys
‚îî‚îÄ‚îÄ contact.ts        #  45+ contact keys

contexts/
‚îî‚îÄ‚îÄ LanguageContext.tsx  # Provider with useLanguage hook
```

### How Translation Works:

1. Import `useLanguage` hook in component
2. Call `t(key)` with translation key
3. System looks up key in `allTranslations`
4. Returns value based on current language
5. Falls back to key if translation missing
